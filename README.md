# PolyAI-EmailSender

## Installation

1. Clone the repo:
```shell
git clone https://github.com/Isaaruwu/PolyAI-EmailSender.git 
```
2. Install the required dependencies:
 ```shell
pip install -r requirements.txt
```
3. Edit SENDER (email) and PASSWORD in .env file
